# eCommerce App

A modern, responsive e-commerce application built with React and Vite. This project offers a seamless shopping experience with product browsing, category filtering, and a fully functional shopping cart.

## Features

- **Dynamic Product Catalog:** Browse products with rich details, including images and localized pricing (EUR).
- **Smart Shopping Cart:** Easily add items, adjust quantities, and track your total item count via the interactive navbar badge.
- **Category Navigation:** Filter products by category to find exactly what you need.
- **Responsive UI:** Built with Tailwind CSS and DaisyUI for a polished look on any device.
- **Client-Side Routing:** Smooth navigation provided by React Router.

## Tech Stack

- **Framework:** React
- **Build Tool:** Vite
- **Styling:** Tailwind CSS, DaisyUI
- **Routing:** React Router
- **Icons:** React Icons

## Getting Started

1.  **Install dependencies:**

    ```bash
    npm install
    npm i react-router
    npm install react-icons --save
    ```

2.  **Run the development server:**
    ```bash
    npm run dev
    ```
